# Enhanced X UP

A raid recruitment addon with player blacklist functionality.

## Description

Enhanced X UP automates raid recruitment by monitoring chat messages and automatically inviting players who send matching keywords. Features a customizable blacklist to filter unwanted players.

## Features

- **Auto-recruitment** - Automatically invite players based on chat messages
- **Flexible filtering** - Match messages by equals, contains, or starts with
- **Chat source selection** - Recruit from all chats, whispers only, or guild chat
- **Player blacklist** - Block specific players from receiving invites
- **Import/Export** - Save and share your blacklist
